## Experience the exciting adventure in Obby: Skateboard Race 

Obby: Skateboard Race features the fast-paced Obby returning for more fun and excitement. This is a fast-paced obstacle course game in which you race skateboards through a challenging world full of twists, turns, and daring stunts. This is a skill test in which you must manage ramps, dodge hazards, and compete to be the first to cross the finish line. This game is ideal for adrenaline addicts and Obby lovers alike, combining racing with innovative platforming for hours of fun and competition!

### Controls

- Use WASD to move
- Press the spacebar to jump
- Use left-click to interact.
