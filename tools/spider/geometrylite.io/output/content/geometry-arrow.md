**Geometry Arrow** is an addicting platform game inspired by Geometry Dash's fast-paced challenges in which you direct a geometric arrow through sophisticated, obstacle-filled courses. 

## Introduce a new Geometry Game - Geometry Arrow    

If you're seeking new challenges in our [Geometry Games](https://geometrylite.io/geometry-games.games) collection, Geometry Arrow is a wonderful option. This is a platform that has rhythm-based gameplay similar to [Geometry Dash Lite](https://geometrylite.io/), however the main character is an arrow. You will control the main character as he moves through a tunnel filled with hazards. Each movement of the arrow generates distinctive waves till it reaches its goal safely. How far can you control the Arrow? Start the game right now and put your reflexes to the test! 

### How to play

- To play, click and hold the LEFT MOUSE BUTTON to control the arrow's upward flight.
- RELEASE the button to make the arrow dive downward.
