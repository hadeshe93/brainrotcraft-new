## **Introduce Geometrical Dash**

Geometrical Dash is an obstacle-course [platform game](https://geometrylite.io/platform.games) with breathtaking thrills. You will embark on a thrilling, heart-stopping journey with a cube. The pace of the round will be uncontrollable, so you must stay focused at every moment. Your goal is to guide the character from the beginning to the end of the map. Of course, no collision is allowed to occur. Even the smallest incident can cause the round to restart.

Geometrical Dash has many challenging levels; each level has two game modes: Normal Mode and Practice Mode. While Practice Mode provides checkpoints for you to continue at the location closest to the collision site, Normal Mode is more severe. However, to be recognized as a true victory, you must pass Normal Mode. Don't worry, because these two game modes in the same level will be exactly the same. You can take advantage of Practice Mode to get familiar with the main map.

### **How to Control**

To start these stunning experiences, customize your cube first. But to be able to unlock new characters, you need to collect bonus coins on the map. This is not a difficult task, so please enjoy it!

To maneuver your cube, you can use the mouse keys or the spacebar. Because the main entity has a somewhat light jump and scholarship So please get used to it and make the appropriate key adjustments.
