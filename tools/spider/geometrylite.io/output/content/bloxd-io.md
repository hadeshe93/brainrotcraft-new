## Play Bloxd.io - A Fun Multiplayer Sandbox Game! 

**Bloxd.io** offers a classic open-world PvP experience! This is a fantastic version of the popular Minecraft game in which you may jump from one side to the other using parkour techniques! There are 18 distinct game modes to choose from, including the well-known survival and creative modes, which players can select according to their interests and abilities.

### Bloxd.io's Key Features 

- Blocky Visual Style: The game, which was heavily influenced by [Minecraft](https://geometrylite.io/tag/minecraft), has straightforward, retro-inspired block-based graphics.
- Multiplayer Action: You can play against gamers from across the world or join forces with pals in this multiplayer game.
- Building Mechanics: You can build anything in this game, from basic buildings to intricate strongholds. Use your creativity and be creative when creating projects!

## How to play

- Use Arrow keys /"WASD" to move 
- Use the Shift key to run
- Use the Mouse to look around
- Use the Left Mouse Click to break a block
- Use Right Mouse Click / E to place a block
- Use Q to drop a block
- Use Tab to open your inventory/craft
- Use the Spacebar to jump
- Use Ctrl, C, or CapsLock to crouch
- Use the Middle Mouse Button or Alt+Left Click to Pick Block (creative)
