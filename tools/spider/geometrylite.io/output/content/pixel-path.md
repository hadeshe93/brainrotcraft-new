## About Pixel Path

Pixel Path features platform gameplay akin to popular games like Geometry Dash and [Devil Dash](https://geometrylite.io/devil-dash). You aim to reach the portal after each level after overcoming a series of obstacles. The issue here is that any obstacle can arise unexpectedly in any location that you cannot predict. It might be deep holes, deadly spikes dropping from the roof, and so on. Each level presents fresh obstacles to you. Try it now and feel the difficulty of the game!

### How to play

To control your character, use the keyboard arrow keys (WASD).
