**Block Blast** is an interesting puzzle game in which you must place blocks in reasonable positions to eliminate lines or columns and score points! 

## About the Gameplay of Block Blast 

Block Blast is a popular puzzle game featuring gameplay that many generations of arcade game fans will recognize. But this game never gets boring because of the challenges that every player aspires to overcome. Your job is extremely simple: destroy all blocks by arranging them in a line. 3 blocks are three options that will appear outside the board; click and drag them into a suitable location to demolish the entire board. 

### How to Get High Scores 

The game rewards instant clearing with large multipliers. Clearing two rows at the same time not only earns you double points but also activates a multiplier, which can significantly boost your score. Make use of this to achieve the greatest possible score.
